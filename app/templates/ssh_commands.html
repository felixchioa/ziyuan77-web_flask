常用SSH命令及其参数

1. ssh

用途：用于远程登录到另一台计算机。

示例：ssh user@hostname

参数：

  • -p [port]：指定连接的端口号。
  • -i [identity_file]：指定私钥文件。
  • -L [local_port:remote_host:remote_port]：设置本地端口转发。

2. scp

用途：用于在本地和远程主机之间安全复制文件。

示例：scp file.txt user@hostname:/path/to/destination

参数：

  • -r：递归复制目录及其内容。
  • -P [port]：指定连接的端口号。
  • -i [identity_file]：指定私钥文件。

3. sftp

用途：用于通过SSH传输文件的交互式文件传输程序。

示例：sftp user@hostname

参数：

  • -P [port]：指定连接的端口号。
  • -i [identity_file]：指定私钥文件。

4. ssh-keygen

用途：用于生成、管理和转换认证密钥。

示例：ssh-keygen -t rsa -b 2048

参数：

  • -t [type]：指定要创建的密钥类型。
  • -b [bits]：指定密钥的位数。
  • -f [filename]：指定密钥文件的名称。

5. ssh-copy-id

用途：用于将本地生成的公钥复制到远程主机的授权密钥文件中。

示例：ssh-copy-id user@hostname

参数：

  • -i [identity_file]：指定要复制的公钥文件。

6. ssh-agent

用途：用于启动一个持有私钥的代理程序。

示例：eval "$(ssh-agent -s)"

参数：

  • -s：生成适用于sh的命令。
  • -c：生成适用于csh的命令。

7. ssh-add

用途：用于将私钥添加到ssh-agent中。

示例：ssh-add ~/.ssh/id_rsa

参数：

  • -l：列出当前ssh-agent中添加的密钥。
  • -d：从ssh-agent中删除指定的密钥。

8. ssh-config

用途：用于配置SSH客户端的选项。

示例：在~/.ssh/config中配置主机别名。

参数：

  • Host：指定主机别名。
  • HostName：指定实际的主机名。
  • User：指定默认用户名。

9. sshfs

用途：用于通过SSH挂载远程文件系统。

示例：sshfs user@hostname:/remote/path /local/mountpoint

参数：

  • -o [option]：指定挂载选项。

10. rsync

用途：用于快速同步文件和目录。

示例：rsync -avz /local/path user@hostname:/remote/path

参数：

  • -a：归档模式，保留文件权限等。
  • -v：详细输出。
  • -z：压缩传输。

11. ssh-keyscan

用途：从远程主机收集公钥。

示例：ssh-keyscan hostname

参数：

  • -t [type]：指定密钥类型。
  • -p [port]：指定端口号。

12. ssh-keysign

用途：用于签署主机密钥。

示例：ssh-keysign

参数：

  • -h：显示帮助信息。

13. sshd

用途：SSH守护进程，用于处理SSH连接。

示例：sshd

参数：

  • -D：不以守护进程方式运行。
  • -p [port]：指定监听端口。

14. ssh-keygen -R

参数：

  • -R [hostname]：指定要删除的主机名。
